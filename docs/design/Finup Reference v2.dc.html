Видалити операцію?
Статус → «видалено». Поза балансом, але залишиться в Журналі як лог.
Поведінка адаптивності
Таблиці → картки
DataTable (desktop) перетворюється на CardList (mobile). Жодного горизонтального скролу.
Меню
Sidebar (desktop) → bottom nav з 4–5 пунктів + sticky FAB (mobile). Імпорт як підменю.
Форми та дії
Поля на всю ширину, sticky primary action унизу. Confirm — inline block або bottom sheet, ніколи прихований.
Доступність
Tap ≥ 44px · фокус-стани · без hover-only · safe-area inset · клавіатура та screen reader.
₴Finup Greenfield · Design Reference v1

Mock-дані, без backend. Операції живуть у Журналі зі статусами pending / approved / deleted; pending та approved впливають на баланс, deleted — ні. Категорія — вільний текст. Світла тема як основна; токени сумісні з темною.
